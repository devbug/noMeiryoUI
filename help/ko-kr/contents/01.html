1. 시작하기에 앞서

1.1 이게 뭐죠?

　이 소프트웨어는 Windows 8/8.1/10의 화면 각 부분의 글꼴을 설정하는 도구입니다.
　어째선지 Windows 8/8.1/10에서 표준 화면의 각 부분의 글꼴을 설정할 수 없게 되었기에 이 프로그램을 만들었습니다.

1.2 작동 환경

　본 프로그램은 Windows 8, 8.1 그리고 10에서 작동합니다.
　데스크탑에 표시된 윈도우의 각 부분의 글꼴을 설정하기 때문에 데스크탑 응용프로그램입니다.

　Windows 2000부터 7까지도 동작은 합니다만, 저 운영체제들은 화면 각 부분의 글꼴을 자체적으로 설정할 수 있기 때문에 의미가 없습니다.
