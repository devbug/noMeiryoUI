2. 사용 방법

2.1 소개

　탐색기에서 noMeiryoUI.exe의 아이콘()을 더블클릭하는 등, noMeiryoUI.exe를 실행합니다.
　아래 화면이 표시됩니다.
　Windows의 버전은 제품명(내부 버전 번호) 형식으로 표시하고 있습니다. 동작 확인, 오류 보고 등에 사용해주세요.


2.2 조작 방법

　화면이 표시되면 화면 각 부분의 명칭과 설정되어 있는 글꼴 이름이 표시됩니다.
　글꼴을 변경하려면 글꼴 이름 옆에 있는 [선택...] 버튼을 누르세요.

　글꼴 선택 대화상자가 표시되면 글꼴, 글꼴 크기, 서식을 취향에 따라 설정해주세요.
　모든 글꼴 설정을 끝내 [확인] 버튼을 누르면, 선택한 글꼴이 화면의 각 부분에 설정됩니다.

　"@"로 시작하는 이름의 세로쓰기용 글꼴은 글꼴 선택 대화상자에서 표시되지 않으며, 따라서 선택할 수 없습니다.

2.3 글꼴 [모두 설정]

　맨 위에 표시되는 [모든 글꼴]이라는 항목은 화면의 각 부분의 글꼴을 모두 같은 글꼴, 글꼴 크기, 서식으로 설정하는 항목입니다.
　글꼴 이름 옆의 [선택...] 버튼을 눌러 글꼴 선택 대화상자에서 글꼴을 선택한 후 [모두 설정] 버튼을 누르면 화면의 각 부분의 글꼴이 모두 같은 글꼴, 글꼴 크기, 서식으로 설정됩니다.

　"@"로 시작하는 이름의 세로쓰기용 글꼴은 글꼴 선택 대화상자에서 표시되지 않으며, 따라서 선택할 수 없습니다.

2.4 글꼴 선택

　글꼴 이름 옆에 있는 [선택...] 버튼을 누르면 아래와 같은 글꼴 선택 대화상자가 표시됩니다.

　글꼴 선택 대화상자에서는 다음과 같은 값을 설정할 수 있습니다.

명칭기능
글꼴 이름(F)글꼴을 지정합니다.
스타일(Y)글꼴의 스타일(굵게, 기울임꼴 등)을 지정합니다.
크기(S)글꼴 크기를 지정합니다.
밑줄(U)체크하면 밑줄이 적용됩니다.
취소선(T)체크하면 취소선이 적용됩니다.
문자 집합(C)문자 집합을 선택합니다.

　글꼴을 선택하고 [확인] 버튼을 누르면 글꼴이 선택됩니다.
　선택 필수 부분이 선택되지 않은 경우 오류 메시지가 표시되므로 지시에 따라 필수 부분을 선택해주세요.

2.5 메뉴

  • 파일(F)
    • 글꼴 설정 열기(O)...
      파일 열기 대화상자가 표시되면 [글꼴 설정 저장(A)...]를 통해 저장한 ini 파일을 선택해주세요.
      파일을 선택하면 [모든 글꼴]을 제외한 [제목 표시줄]부터 [메뉴]까지의 항목의 글꼴 설정이 ini 파일의 설정으로 지정됩니다.
    • 글꼴 설정 저장(A)...
      선택한 글꼴 설정을 ini 파일에 저장합니다.
      저장되는 설정은 [모든 글꼴]을 제외한 [제목 표시줄]부터 [메뉴]까지의 항목의 글꼴 설정입니다.
      파일 저장 대화상자가 표시되면, 확인하기 쉽게 파일 이름에 글꼴 이름을 붙여 저장해주세요.
    • 시스템 글꼴 설정 및 종료(S)
      선택한 글꼴을 화면 각 부분에 설정하고 프로그램을 종료합니다.
    • 종료(X)
      글꼴 설정을 적용하지 않고 프로그램을 종료합니다.
  • 프리셋(P)
    • Windows 8/8.1
      [모든 글꼴] 이외의 설정을 Windows 8/8.1의 기본값으로 되돌립니다.
    • Windows 10
      [모든 글꼴] 이외의 설정을 Windows 10의 기본값으로 되돌립니다.
  • 도구(T)
    • 개별 스레드에서 시스템 글꼴 설정(A)
      특정 상황 하의 트러블슈팅 용 설정을 수행합니다.
      SetPoint(Logicool 마우스/키보드 유틸리티) 등을 실행한 환경에서 글꼴을 설정했을 때 '응답 없음'이 되는 경우, 이 메뉴를 체크하고 글꼴을 설정해주세요.
      이 메뉴는 기본적으로 체크되어 있습니다.
    • Windows 7 방식의 글꼴 크기 계산(7)
      글꼴 크기의 계산을 Windows 7과 같은 방식으로 합니다.
      Windows 8/8.1에서 Windows 7 이전과 동일한 글꼴 크기를 지정하고 싶을 경우 이 옵션을 사용합니다.

      이 옵션을 체크 해제한 상태에서 글꼴 크기에 11포인트 이상을 지정한 경우, 글꼴 크기는 Windows 8/8.1의 디스플레이 설정의 텍스트 크기와 같은 크기가 됩니다. 이 경우, 글꼴 크기에 따라 Windows 7 이전의 텍스트 크기 값보다 1픽셀 커집니다.
      이 옵션을 체크한 후 글꼴을 설정하면 Windows 8/8.1이라도 글꼴의 크기가 Windows 7 이전과 같은 크기가 됩니다.
      이 옵션을 사용하여 글꼴 크기를 지정하는 것은 이 옵션을 설정한 후에 선택한 글꼴 설정에만 유효합니다.
      Windows 7 이전의 OS와 Windows 10에서 이 옵션은 비활성화되고, 선택 불가능하게 됩니다. 글꼴 크기 계산은 Windows 7 이전 OS에 준합니다.
  • 도움말(H)
    • 도움말(H)
      이 문서를 열어 화면에 표시합니다.
    • No!! Meiryo UI에 대하여(A)...
      본 프로그램의 버전 대화상자을 표시합니다.

2.6 명령줄 옵션

　명령줄 옵션으로 실행 시에 글꼴 설정을 읽을 수 있고, 지정한 글꼴 설정을 자동으로 시스템 글꼴로 설정하는 것이 가능합니다.

　명령줄은 다음과 같은 형식을 따릅니다.
noMeiryoUI 글꼴 설정 파일 이름 [-set]

　글꼴 설정 파일은 이 프로그램에서 글꼴 설정을 저장한 ini 파일을 의미합니다.
　글꼴 설정 파일 이름에 공백이 있을 경우 파일 이름을 "(쌍따옴표)를 이용해 묶어주세요.

　글꼴 설정 파일 이름 뒤에 공백을 넣고 -set라고 입력하면 지정한 글꼴 설정 파일의 내용으로 시스템 글꼴을 설정하고 프로그램을 종료합니다.
배치 파일 등에 사용하면 미리 준비한 글꼴 설정 파일의 내용으로 시스템 글꼴을 한번에 설정하는 것이 가능해 편리합니다.

예1. MySet.ini라는 글꼴 설정 파일을 실행 시 읽는 경우

　MySet.ini라는 글꼴 설정 파일을 실행 시에 읽는 경우 명령줄 등에 다음과 같이 입력합니다.
noMeiryoUI MySet.ini

예2. Your Settings.ini라는 글꼴 설정 파일의 내용으로 시스템 글꼴을 설정하는 경우

　Your Settings.ini라는 글꼴 설정 파일에 따라 시스템 글꼴을 설정하는 경우 명령줄 등에 다음과 같이 입력합니다.
noMeiryoUI "Your Settings.ini" -set
