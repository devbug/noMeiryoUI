3. 제작 이력

2017/06/03 Version 2.34

  • 실행할 때 프로그램을 마우스 포인터가 있는 모니터의 중앙보다 약간 왼쪽 위에 표시하도록 했습니다.
  • 글꼴 선택 대화상자를 프로그램 메인 창의 중앙에 표시하도록 했습니다.
  • 도움말 파일을 프로그램 메인 화면이 표시되어있는 모니터의 중앙보다 약간 왼쪽 위에 표시하도록 했습니다.
  • 중국어(간체)판 도움말의 글꼴이 일부 다른 글꼴로 표시되는 문제를 수정했습니다.

2017/05/14 Version 2.33

  • Windows 10에서 실행했을 때 상세한 버전 정보를 표시하도록 했습니다.
  • 영문판 Windows에서도 프리셋 메뉴가 기능하도록 했습니다.
  • 각 언어로의 번역에 대비해, 표시 글꼴과 프리셋의 문자 집합을 변경할 수 있도록 했습니다.
  • 읽어들일 언어 파일, 도움말 파일 이름을 Windows의 로캘 정보로부터 결정하도록 했습니다. 따라서 번역을 추가할 때 프로그램을 변경할 필요가 없고, 언어 파일만 배포하면 되게 되었습니다.

2016/10/10 Version 2.32

  • 타치바나 겍카님의 도움으로 도움말의 중국어(간체) 버전을 포함하고, 중국어(간체)판 Windows에서 실행할 때 표시할 수 있도록 했습니다.
  • 중국어(간체)판 Windows에서도 프리셋 메뉴가 기능하도록 했습니다.
  • 각 언어로의 번역에 대비해, 소스코드 폴더 구조를 변경하고 메시지를 기술한 파일과 도움말 파일의 번역 방법에 대해 간단한 설명을 추가했습니다.

2016/09/19 Version 2.31

  • 타치바나 겍카님이 중국어(간체)로 번역해주셨습니다. 이제 중국어(간체)판 Windows에서 중국어(간체)로 표시됩니다.
  • 도움말의 국제화에 대비해, 도움말을 HTML로 변경했습니다.
  • 버전 정보에 번역자의 정보를 추가했습니다.
  • 일본어판 이외의 Windows에서 버전 정보를 표시했을 때 발생하는 잠재적 문제를 수정했습니다.

2016/08/19 Version 2.30

  • 일본어 이외의 OS를 위한 다국어 지원이 추가되었습니다. 이 버전에선 일본어, 중국어(간체), 그 외 언어로 메시지가 변경됩니다.
    단, 저는 중국어를 모르기에 이 버전에선 임시로 중국어(간체)판 Windows에서 실행할 때 영어로 표시됩니다.

2016/06/09 Version 2.20

  • 명령줄로부터 설정 파일을 지정할 수 있도록 했습니다. 또한, 명령줄로부터 지정한 설정 파일의 내용에 글꼴을 설정할 수 있도록 했습니다.

2016/02/23 Version 2.17
